KamilleGame2

Ein kleines Java-Konsolenspiel mit einer „Liebesblume“.
Der Benutzer wählt ein Blütenblatt aus und erhält ein zufälliges Ergebnis wie in einem klassischen Kamillenspiel.

Funktionen
Auswahl eines Blütenblatts von 1 bis 6
Zufällige Vorhersagen
Eingabeprüfung
Mehrere Spielrunden möglich
Konsolenbasierte Benutzereingabe
Mögliche Ergebnisse

Das Spiel kann folgende Vorhersagen anzeigen:

Liebt
Liebt nicht
Spuckt
Küsst
Heiratet
Schickt in die Hölle
